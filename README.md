# revealjs-de-an

Reveal.js presentation scaffold based on `/home/hung/Downloads/revealjs_presentation-main`.

## Structure

- `index.html`: main presentation file
- `figures/`: image assets used by the slides
- `presentation_script.md`: speaker script/content notes
- `template/`: reference PPT/PDF template assets

## Run locally

From the project root:

```bash
npm start
```

Then open `http://localhost:8000`.
